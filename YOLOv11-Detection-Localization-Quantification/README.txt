// Training instructions

The current code requires the model to be run twice, at first run 40 epoch training is done and then in the next run 20 epoch more is trained.

Code modifications may be required to run these two steps. By simply uncommenting and commenting the code sections of respective run.

//  Dataset requirement

This particular model is trained using the 'GRAZPEDWRI-DX' dataset available in Kaggle: https://www.kaggle.com/datasets/jasonroggy/grazpedwri-dx
