// This folder contains the model deployment script. 

By running the python code app.py, you can locally host the project model and test in a live demonstration.

1. Type 'python app.py' in the terminal
2. Copy the link provided.
3. Paste the address and press enter in a browser of your choice.
4. Upload an image (Pediatric Wrist bone X-ray) and press 'Analyze Image'.
